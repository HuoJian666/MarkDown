## 列表
### 无序列表

就是\*加空格
* name:huo
* age:20
* school：烟台

 
二级列表：前面加上两个空格再写，三级列表四个空格再写，以此类推
- name:huo 
  - major：software 
    - address： 哈哈哈  
    
### 有序列表
数字加.(点)加空格  
1. name:huo    
    2. qqqqqP(前面打了四个空格)
2. age:20  
4. 想在2，3之间插入一行，注意它会重新排列序号
3. school:烟台
